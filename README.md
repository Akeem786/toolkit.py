# 🛠 Python Penetration Testing Toolkit

A modular penetration testing toolkit with multiple security tools.

## 📌 Features
✅ **Port Scanner** - Detect open ports on a target  
✅ **Brute Force Attack** - Test weak credentials  
✅ **Subdomain Enumeration** - Find subdomains of a domain  
✅ **Web Vulnerability Scanner** - Detect SQL Injection & XSS
